# Result files

- `paper_benchmark_table3.csv`: published WAPE/WPE values used in the notebooks.
- `our_paper_comparable_summary.csv`: headline XGBoost metrics on uncensored target days.
- `full_benchmark_ranking.csv`: combined ranks by WAPE and absolute WPE.
- `pairwise_comparison_vs_each_paper_model.csv`: differences versus every paper model.
- `mandatory_vs_zero_start.csv`: direct XGBoost ablation and relative reductions.
- `our_evaluation_all_subsets_and_horizons.csv`: all groups, subsets, and horizons.
- `horizon_comparison_uncensored_overall.csv`: day-by-day XGBoost comparison.
- `fresh_temporal_cv_summary.csv` and `fresh_temporal_cv_folds.csv`: post-selection CV.
- `feature_selection_path.csv`: accepted and rejected selection steps.
- `selected_features_and_interpretation.csv`: selected variables and caveats.
- `hyperparameters.csv`: initial and final tuned settings.
- `run_manifest.json`: machine-readable scope and headline claims.
